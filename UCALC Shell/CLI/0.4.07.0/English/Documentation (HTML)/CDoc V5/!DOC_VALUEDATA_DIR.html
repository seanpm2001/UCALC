ValueData Directory

About

The ValueData folder is a list of common numbers that serve as a basic reference for numbers. The files are not used by the system and can only be opened with text editors

Removing the folder wouldn't help, as the folder is less than 1 and a half kilobytes compared to a multi-megabyte program.

INTEGER files

These files in the ValueData folder are based off Python Syntax. However, they cannot be run in Python without heavy modification. They contain numbers with no decimals, and are the most common file type in this directory, with 34 in existance

FLOAT files

Since these files in the ValueData folder are based off Python Syntax, they are mimicked. The name "FLOAT" comes from Pythons value of storing numbers with decimals. In this directory, they are the rarest with only 7 in existance. They still cannot be run in Python wuthout heavy modification and are designed to be run in a text editor
