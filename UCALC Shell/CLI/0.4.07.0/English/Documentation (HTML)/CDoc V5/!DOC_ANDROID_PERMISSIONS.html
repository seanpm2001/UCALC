Android Permission Info

Required permissions

For normal use

Common required permissions are non-existant. There are no required permissions for vanilla usage

For Real-mode calculation

Real-mode calculation a couple of permissions. Without them, the feature wouldn't work

Required permissions:

<*> Allow UCALC to access your camera

<*> Allow UCALC to access this devices storage

Not very many permissions are required. These are standard for this mode

For accessibility features

There is only 1 permission required for accessibility features:

<*> Allow UCALC to acces the microphone

The Microphone will only be un use when the app is open and using the text-to-speech or voice-based calculation modes. If the Microphone is still in use after the app is closed, you may have a malicious version

Banned permissions

UCALC uses some permissions for its many features, but there are some permissions you should NEVER see or agree to.

Some possible remixes/distros can contain malicious activity. Here are the permissions that you should never see:

<*> Send SMS messages

<*> Read SMS messages

<*> Make/manage phone calls

<*> Identity

<*> Location

<*> Access contacts

<*> Gain SuperUser Access
