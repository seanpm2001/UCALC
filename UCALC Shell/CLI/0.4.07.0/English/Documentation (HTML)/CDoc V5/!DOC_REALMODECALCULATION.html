Real-time calculation mode

Features

Real-time calculation mode is a mode where you can take a picture of math equations and it will automatically solve them for you.

It doesn't stop there. You can do so much with it! You can:

* Export as a document

* Export as a picture

* Print it out

* Make it into a video

* Translate word problems

* Make an equation library

* Operate a terminal

Export formats

Pictures

Pictures can be exported in many formats. Here is a list of supported formats:

.JPG

.PNG

.HEIF

.GIF

.TIF

.BMP

Documents

Documents can also be exported in many formats. Here is a list of supported formats:

.EPUB (Version 3.0)

.XPS

.ODT

.UOT

.TXT

Resolution

You can choose the resolution of the camera. The lowest quality you can go down to is 1280x720.

Feature compatibility

This is a big feature that works very differently from the rest of the calculator. Unfortunately, this means that the calculator isn't fully compatible on some systems, as this feature requires more modern devices than the bare minimum for other features

Android support

The first version of Android that can run this feature is 6.0 Marshmallow

Windows support

The first version of Windows that can run this feature is Windows Vista

MacOS support

The first version of MacOS that can run this feature is OX X 10.10 (Yosemite)

Debian support

Ubuntu

The first version of Ubuntu that can run this feature is Ubuntu 10.10 (Maverick Meerkat)

Lubuntu

The first version of Lubuntu that can run this feature is Lubuntu 10.10

Red Hat Linux support

Red Hat Enterprise

Red Hat Linux is NOT supported. However, some Distros based on Red Hat are

Fedora Linux

The first version of Fedora that can run this feature is Fedora 14

Helpful tips and guidelines

This feature is NOT designed to cheat on your math homework or schoolwork. Teachers can use this feature to make answersheets, but Students shouldn't use this as a cheat sheet. Math is very important and shouldn't be cheated on

It is recommended to clean your camera before taking pictures to get better pictures and have more accuracy

./limit | Set library limits |
