Requirements

RAM

UCALC doesn't require high system requirements. You should easily be able to run it with 128 megabytes of RAM, but it might go slow. To have perfect speed, use 4 gigabytes of RAM or higher

Resolution

UCALC is designed to be run on computers with a resolution of 1280x720 or higher. It can be run all the way down to 360x240 but it won't fit well on the screen

Internet

UCALC does NOT require an internet connection. It is designed to run offline
